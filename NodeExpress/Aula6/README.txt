-- HTTP REQUEST --

O Request é uma forma de requisitar algo para o servidor, para assim, receber um response, ou seja, uma resposta.

Utilizando Insominia.

Nos Headers é possível definir diversos status para seu servidor, inclusive colocar a versão do back-end que você está utilizando.

Nesse link está os protocolos usados no HTTP e seus requests possíveis.
https://developer.mozilla.org/pt-BR/docs/Web/API/Request

O body é para onde vai o que foi requisitado, por exemplo, você requisita os filmes de terror de um catálogo da Netflix e ele puxa do back-end.
