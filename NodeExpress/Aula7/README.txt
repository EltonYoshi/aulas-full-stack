-- HTTP RESPONSE --

O response possui 3 partes, sendo eles: status (versão HTTP, código do Status e Mensagem).

No Insomnia a parte verde é a resposta, a amarela a requisição e a rosa o body

Os códigos de Status é tipo o famoso código 404 de ERRO e sendo 200 o de OK.
    Cada código possui seu objetivo próprio e é consultá-los na hora de criar um response.
